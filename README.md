# window_setting

## 테마 설정
1. .mytheme.omp.json을 홈디렉토리(C:\\Users\\[User-name])으로 복사 
2. 이후 oh-my-posh에서 사용하려는 테마의 내용을 .mytheme.omp.json에 덮어쓰기
3. powershell 재실행
